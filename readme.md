# Next.js Project Test Amerta Arya Disastra

Project ini dibuat menggunakan **Next.js**.

## Requirements

disini saya menggunakan : 
1. mysql
2. node v24.12.0
3. npm versi 11.6.2

# Cara Menjalankan
1. restore database dari sql yang ada di root folder namanya "amerta_test_aryadisastra.sql"
2. setting .env.local sesuai environtment local pc/laptop
3. install depedencies dengan npm install
4. jalankan npm run dev

## *notes : root project folder ada di "/next-js-version"
